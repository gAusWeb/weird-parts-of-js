Greeting - Custom Mini Framework / Library
-------------------------------------------

What does it do?
- When given a first name, last name, and optional language, it generates formal and informal greetings.
- Support English and Spanish languages
- Reusable libray / framework
- East to type 'G$()' structure (just like jQuery's '$' usage)
- This will also support jQuery - Ability to pass a jQuery object that points at a html element, that will fill the element with my greeting
